Neptune Splash API

Neptune Splash API is my second Minecraft "god mod". It does two things at once - a mod to allow users to define their own splash texts, and an API to allow mod devs to add their own splashes to the main menu. Learn more here.


Pluto

Pluto is the first of my Minecraft "god mods". It automatically enchants golden tools on right-clicking, thereby making golden tools worthwhile once more. It also enables Golden Pickaxes to mine diamonds. Learn more here.
